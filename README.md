# coatless-rust.github.io

Auto-built project index for [coatless-rust](https://github.com/coatless-rust), deployed to
[rust.thecoatlessprofessor.com](https://rust.thecoatlessprofessor.com/).

Source: [coatless/quarto-org-index](https://github.com/coatless/quarto-org-index).

Rebuilt daily by GitHub Actions plus on push to `main` and on
`repository_dispatch` event `refresh`.
